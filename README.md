# StrataBayes

In this repository, we provide the code for the simulation study and application on social mobility of the paper <a href=https://arxiv.org/abs/2412.00311>_"Disentangling The Effects of Air Pollution on Social Mobility: A Bayesian Principal Stratification Approach"_ </a> by D. Zorzetto, P. Dalla Torre, S. Petrone, and F. Dominici, F.J. Bargagli-Stoffi
